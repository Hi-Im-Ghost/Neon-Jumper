\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \mbox{\hyperlink{classrapidjson_1_1_allocator}{Allocator}} = Crt\+Allocator$>$\newline
struct Generic\+Memory\+Buffer$<$ Allocator $>$}
Represents an in-\/memory output byte stream. 

This class is mainly for being wrapped by \mbox{\hyperlink{class_encoded_output_stream}{Encoded\+Output\+Stream}} or \mbox{\hyperlink{class_auto_u_t_f_output_stream}{Auto\+UTFOutput\+Stream}}.

It is similar to File\+Write\+Buffer but the destination is an in-\/memory buffer instead of a file.

Differences between Memory\+Buffer and String\+Buffer\+:
\begin{DoxyEnumerate}
\item String\+Buffer has Encoding but Memory\+Buffer is only a byte buffer.
\item \mbox{\hyperlink{class_generic_string_buffer_a42079865bab1deb930c6cff7d491c719}{String\+Buffer\+::\+Get\+String()}} returns a null-\/terminated string. \mbox{\hyperlink{struct_generic_memory_buffer_a65217e77ae6d4c249c16360c24854411}{Memory\+Buffer\+::\+Get\+Buffer()}} returns a buffer without terminator.
\end{DoxyEnumerate}
