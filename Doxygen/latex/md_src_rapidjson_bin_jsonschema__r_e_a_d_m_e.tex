This repository contains a set of JSON objects that implementors of JSON \mbox{\hyperlink{class_schema}{Schema}} validation libraries can use to test their validators.

It is meant to be language agnostic and should require only a JSON parser.

The conversion of the JSON objects into tests within your test framework of choice is still the job of the validator implementor.\hypertarget{md_src_rapidjson_bin_jsonschema__r_e_a_d_m_e_autotoc_md1}{}\doxysection{Structure of a Test}\label{md_src_rapidjson_bin_jsonschema__r_e_a_d_m_e_autotoc_md1}
If you\textquotesingle{}re going to use this suite, you need to know how tests are laid out. The tests are contained in the {\ttfamily tests} directory at the root of this repository.

Inside that directory is a subdirectory for each draft or version of the schema. We\textquotesingle{}ll use {\ttfamily draft3} as an example.

If you look inside the draft directory, there are a number of {\ttfamily .json} files, which logically group a set of test cases together. Often the grouping is by property under test, but not always, especially within optional test files (discussed below).

Inside each {\ttfamily .json} file is a single array containing objects. It\textquotesingle{}s easiest to illustrate the structure of these with an example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\{}
\DoxyCodeLine{    "{}description"{}: "{}the description of the test case"{},}
\DoxyCodeLine{    "{}schema"{}: \{"{}the schema that should"{} : "{}be validated against"{}\},}
\DoxyCodeLine{    "{}tests"{}: [}
\DoxyCodeLine{        \{}
\DoxyCodeLine{            "{}description"{}: "{}a specific test of a valid instance"{},}
\DoxyCodeLine{            "{}data"{}: "{}the instance"{},}
\DoxyCodeLine{            "{}valid"{}: true}
\DoxyCodeLine{        \},}
\DoxyCodeLine{        \{}
\DoxyCodeLine{            "{}description"{}: "{}another specific test this time, invalid"{},}
\DoxyCodeLine{            "{}data"{}: 15,}
\DoxyCodeLine{            "{}valid"{}: false}
\DoxyCodeLine{        \}}
\DoxyCodeLine{    ]}
\DoxyCodeLine{\}}

\end{DoxyCode}
