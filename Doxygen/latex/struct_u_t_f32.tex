\begin{DoxyTemplParams}{Template Parameters}
{\em Char\+Type} & Type for storing 32-\/bit UTF-\/32 data. Default is unsigned. C++11 may use char32\+\_\+t instead. \\
\hline
\end{DoxyTemplParams}
\begin{DoxyNote}{Note}
implements Encoding concept

For in-\/memory access, no need to concern endianness. The code units and code points are represented by CPU\textquotesingle{}s endianness. For streaming, use \mbox{\hyperlink{struct_u_t_f32_l_e}{UTF32\+LE}} and \mbox{\hyperlink{struct_u_t_f32_b_e}{UTF32\+BE}}, which handle endianness.
