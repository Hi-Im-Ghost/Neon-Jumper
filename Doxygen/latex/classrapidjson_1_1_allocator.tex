\doxysubsection{Detailed Description}
Concept for allocating, resizing and freeing memory block. 

Note that \mbox{\hyperlink{allocators_8h_a7e0ace0214f18427d9ee8e76a5b3dcdf}{Malloc()}} and \mbox{\hyperlink{allocators_8h_a8e4137cbc097868e8c3d785f6cca1223}{Realloc()}} are non-\/static but \mbox{\hyperlink{allocators_8h_a471c182d62d396b7d5d564e8d6a62d9e}{Free()}} is static.

So if an allocator need to support \mbox{\hyperlink{allocators_8h_a471c182d62d396b7d5d564e8d6a62d9e}{Free()}}, it needs to put its pointer in the header of memory block.
